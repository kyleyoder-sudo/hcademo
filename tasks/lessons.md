# Lessons

## Visual Direction Corrections
- When a user asks for a clean presentation site, default to the most minimal interpretation first.
- Reduce copy harder than feels necessary on demo-facing pages; keep only headline, short support line, and primary CTA unless more information is essential.
- Prefer sharper card geometry and simpler composition over soft, layered “designed” surfaces when the user says the site still feels busy.
- If the first redesign still includes multiple explanatory text blocks, assume another pass should remove sections before adding styling refinements.
- If the user says they dislike the entire direction, stop iterating within that design family and pick a genuinely new art direction with a different mood, palette, and composition language.
